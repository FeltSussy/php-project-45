A five-game console collection modeled on popular brain-training apps. Answer questions correctly in each game. Three right answers win the round. One wrong answer ends it, forcing a restart.


## Prerequisites


* Linux, Macos, WSL
* PHP >=8.0
* Make
* Git


## Setup

Setup [SSH](https://docs.github.com/en/authentication/connecting-to-github-with-ssh) before clone:

```bash
git clone git@github.com:FeltSussy/php-project-45.git
cd php-project-45

make install
```

## Run game

"Calculator"

```bash
make brain-calc
```
